Operating System Fundamentals

    A Linux distribution (distro) is an operating system that usually includes:
  1. the Linux kernel,
  2. GNU tools and libraries,
  3. added software, documentation,
  4. a window system (the most common being the X Window System, or, more recently, Wayland),
  5. a window manager,
  6. and a desktop environment.
Linux users usually obtain their operating system by downloading one of the Linux distributions.
    Some common distributions are:
  • CentOS
  • Red Hat® Enterprise Linux (RHEL)
  • Fedora
  • SUSE Linux Enterprise
  • openSUSE
  • Debian
  • Ubuntu
  • Mandriva
  • Mint
Linux distributions are like each other, but each has its own strengths and weaknesses. The software packaged with each distribution can make a huge difference in how Linux works for you. Although you can always download or buy missing components, the software package is easier to use if the components have already been tested and compiled together to work with your distribution.

Getting Help & Man Pages in Linux

In Linux, getting help and using man pages (manual pages) is essential for understanding and using various commands and utilities. Man pages provide detailed information about commands, their options, and how to use them.

man Command

The man command is used to access the manual pages for various commands. Syntax: man [options] command Example: man ls The manual page will be displayed in your terminal, allowing you to navigate through it using the arrow keys. Press q to exit the manual page viewer.

--apropos

Search for commands related to a keyword. Syntax: command --apropos

--whatis

Display a one-line description of a command. Syntax: command --whatis

--help

the --help option. Syntax: command --help Example: ls --help This will display information about the command's usage, available options, and arguments.

Info

Some commands have more comprehensive documentation in the Info format, which can be accessed using the info command. Syntax: info command Example: info coreutils

linux commands

whoami

The whoami command is used to display the username you have used to log in to the system. Sometimes, you may need to log in to a system and switch among different users. If you lose track of which account you are logged into, use the whoami command.

hostname

The hostname command is used to display the hostname of the system you are currently logged in to. When you log in to different systems using the same terminal, you can use the hostname command to find the system on which you are presently running the command.

date

The date command displays the current date and time set on a system. The syntax of the date command is date +[format], where format is the string of characters that are used to display the different fields of the output.

The syntax of the date command is date +[format]
Character Format Description
%d Displays the current day of the month (01 to 31).
D % Displays the date in the format mm/dd/yy, where mm is month, dd is day, and yy is year.
%H Displays the current hour in the 24-hour format (00 to 23).
%l Displays the current hour (1 to 12). This choice does not display A.M. or P.M. after the hour.
%m Displays the current month of the year (01 to 12).
%M Displays the current minute (00 to 59).
%r Displays the time in the 12-hour format; i.e., hh:mm:ss [A.M. or P.M.].
%R Displays the time in the 24-hour format; i.e., hh:mm. This choice does not display the seconds.
%S Displays the seconds (00 to 60).
%T Displays the time in the 24-hour format; i.e., hh:mm:ss. This choice displays the seconds also.
%y Displays the last two digits of the current year.
%Y Displays the current year in four digits(yyyy).
hints can be displayed by command: man date or date --help(brief help)

uptime

The uptime command displays the time from when a system started running. The output of the uptime command gives information about the current time, how long the system is running, and how many users are currently logged in.

who

If you’re working on a server, there may be multiple users logged in at the same time. The who command is used to figure out the details of users currently logged in to a system. The output of the who command includes the username, the name of the system from which the user is connected, and the time since the user is connected.

w

The w command is primarily used to display the details of users who are currently logged in to a system and their transactions. The first line of the output displays the status of the system. The second line of the output displays a table with the first column listing the users logged in to the system and the last column showing the current activities of the users. The remaining columns of the table show different attributes associated with the users.

last

The last command displays the history of user log in and log out, along with the actual time and date. It also has options that enable you to filter users who have logged in through a specific terminal. The last command retrieves information from /var/log/wtmp file.
